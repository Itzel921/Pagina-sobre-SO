 Los lenguajes de programación orientados a objetos tratan a los programas como conjuntos de objetos que se ayudan entre ellos para realizar acciones. Entendiendo como objeto al entidades que contienen datos. Permitiendo que los programas sean más fáciles de escribir, mantener y reutilizar.

Fue diseñado por los profesores Ole-Johan Dahl y Kristen Nygaard en Noruega. Este lenguaje de programación orientado a objetos fue el “Simula 67” que fue un lenguaje creado para hacer simulaciones de naves.Esta programación orientada a objetos (POO) tomo auge a mediados de los años

Lenguajes de programación orientados a objetos

Si bien Simula se acredita como el primer lenguaje de programación orientado a objetos, los lenguajes de programación orientada a objetos más populares son:
  • Java

  • C++
  • Ruby
  • Scala
  • PHP
  • C#
  • VB.NET
  • Clarion
  • Eiffel
  • PowerBuilder
